## 概要

Sora のウェブフックを処理するサーバーアプリケーションを開発するための SDK です。

### 対応内容

- [x] AuthWebhook
- [x] SessionWebhook
- [x] EventWebhook
- [ ] API 

## 利用方法

v2022.2 は対応する Sora のバージョンを表しています。

```
go get -u github.com/shiguredo/sora-app-sdk-go/v2022.2
```
